Solve

Los botones de la parte inferior derecha sirven para lanzar el algoritmo con posibles opciones. Si la checkbox de Time está activa, se cronometrará el tiempo tardado para generar la habitación. Si la opción Debug está activada se abrirá el debugger. Una vez se ha encontrado solución, si se vuelve a accionar el botón de 'Solve Design', el algoritmo continuará desde donde lo dejo.
