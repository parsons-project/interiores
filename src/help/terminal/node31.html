add

Sintaxis
add constraint <tipo> <params> <selected1> [<selected2>]
Descripción
Añade una restricción de tipo tipo con los parametros params. La restricción puede ser unaria o binaria. Las unarias incluyen: 'color', 'model', 'price', 'material', 'orientation', 'width', 'depth' y 'position'. La binarias pueden ser de distancia máxima o mínima. Las restricciones se aplican sobre elementos seleccionados (tienen la forma tiposNUMERO.
