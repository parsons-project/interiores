add

Sintaxis
add gc <tipo> <params>
Descripción
Añade una restricción global (para toda la habitación). Esta restricción puede ser o bien de presupuesto máximo (budget), que todos tengan el mismo color o el mismo material (same-color, same-material) o bien que todos tengan un color o un material especificados (specific-color, specific-material).
