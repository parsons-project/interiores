Debugger

Esta ventana permite ejecutar el algoritmo con control absoluto. Se muestra el valor que tiene cada variable en cada momento. Se muestran las posiciones que prueba cada mueble (con colores distintos para cada uno). Se puede ejecutar el algoritmo a tramos con 'Resume'/'Pause' y parar la ejecución con 'Stop' o bien iterar paso a paso indicando el numero de iteraciones que debe hacer y pulsando 'Iterate'.
