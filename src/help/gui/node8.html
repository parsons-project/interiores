Room types catalog

Abre el editor de catálogos de tipos de habitaciones. En él se puede crear nuevos tipos y modificar o eliminar ya existentes. Se piden parámetros necesarios para el programa como tamaño mínimo y máximo, elementos prohibidos y obligatorios... Pueden crearse nuevos catálogos y guardarlos en disco para posteriormente recuperarlos.
